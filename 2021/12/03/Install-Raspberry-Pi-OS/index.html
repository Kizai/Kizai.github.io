树莓派(Raspberry Pi)3B+系统安装与环境搭建(详解)

Raspberry Pi 3 Model B+

一、前期准备:

  • Raspberry Pi 3B+
  • SD card :推荐使用闪迪Class10 16G+的卡。
  • 磁盘映像工具win32diskimager-1.0.0-install.exe
  • SD卡格式化工具SD Card Formatter 5.0.1 Setup.exe
  • MobaXterm

二、系统下载：

  • Raspberry Pi OS
  • 可以根据自己使用的需求下载你所需要版本，推荐下载种子的方法用迅雷下载。

三、格式化SD卡

  1. 点击下载好的SD Card Formatter.exe,插入SD卡到电脑上。
  2. 选中你的SD卡，点击Format，等待进度条完成就代表格式化成功啦。

四、写入系统镜像到SD Card

  1. 点击win32diskimager.exe打开磁盘映像工具。
  2. 解压下载好的系统镜像压缩包。

  1. 选择刚刚解压好的系统镜像压缩包和SD卡驱动,点击开始写入，会有弹窗警告，点击确认，等待进度条完成(右下角会有预计总时间)。

  1. 点击进入boot根目录，使用Git bash窗口创建ssh文件，没有任何后缀的，为下一步做好准备。

    五、通过网线连接树莓派继续配置

  2. 这个过程需要你的电脑是有wifi功能或者USB网卡。
  3. 把SD插回到树莓派3B+,上电开机,用一根网线把电脑和树莓派连在一起。
  4. 进入系统设置->打开更改适配器选项->选择你的wifi或者无线网卡的wifi->双击点击属性->共享->[x]允许其他用户xxxx->以太网->确定。


  1. 双击以太网->属性->双击(TCP/IPv4)->改成使用下面的ip地址。


  1. 使用win+R输入cmd,进入终端，输入arp -a,找到物理地址以b8开头的ip地址。

  1. 输入ssh pi@192.168.137.xxx根据自己树莓派的ip地址进行输入，第一次连接会提示是否连接，直接输入yes即可。
  2. 树莓派的默认密码是raspberry，输密码是看不到输入，输进去点击ENTER即可连接到树莓派上。
  3. 看到下图就证明你成功连上树莓派啦！！！

配置树莓派

更换树莓派的镜像源：

  1. 备份原源文件
  2. 修改source.list文件，把之前的源注释掉，换成阿里云的源。
  3. 更新软件包。

配置远程桌面连接
